# O2 Heater Circuit

the O2 heater circuit is what makes 4-wire [Oxygen Sensor](/cars/electronics/oxygen-sensor)s different from 1-wire sensors. The heater circuit helps the sensor stay in its optimum temperature range so that it reads accurately. See [One Wire To4 Wire O2 Sensor](/cars/electronics/one-wire-to4-wire-o2-sensor) for information on adding circuitry to an [ECU](/cars/electronics/ecu) to support the [O2 Heater Circuit](/cars/electronics/o2-heater-circuit). See the [Rom Maps](/cars/electronics/rom-maps) for information on diabling the code 41 present when it is absent/malfunctioning.
