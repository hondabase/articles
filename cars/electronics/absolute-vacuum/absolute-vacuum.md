# Absolute Vacuum

Absolute vacuum only truly exists in deep space. It is the state where an object is surrounded by absolutely nothing. There is absolutely zero gas pressure in an absolute vacuum. You can only approach an absolute vacuum here on earth, but it is a very important concept for things like [Absolute Pressure](/cars/electronics/absolute-pressure)
