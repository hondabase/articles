# ECU-Code Compatibility

Most honda [ECU](/cars/electronics/ecu)s can run code from other [ECU](/cars/electronics/ecu)s of a similar age, and similar family. Civics and Integras generally can share code. Preludes and Accords are certainly from the same design team, and sometimes can share code. For specific information consult:

- [OBD0 Code Compatibility](/cars/electronics/obd0-code-compatibility) - addresses [OBD0 MPFI](/cars/electronics/obd0mpfi) [ECU](/cars/electronics/ecu)s
- [OBD1 Code Compatibility](/cars/electronics/obd1-code-compatibility) - addresses [OBD1 Civic Integra](/cars/electronics/obd1-civic-integra) [ECU](/cars/electronics/ecu)s
