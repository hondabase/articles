# Serial Adapter

All Honda [ECU](/cars/electronics/ecu)s use 5v [TTL](/cars/electronics/ttl) signaling for [serial communication](/cars/electronics/serial-communication). In order to connect to an [ECU](/cars/electronics/ecu), if the hardware used for [Data Logging](/cars/electronics/data-logging) doesn't support this, you must use some form of serial adapter.
