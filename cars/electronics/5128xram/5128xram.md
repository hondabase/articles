# 5128XRAM

5128 XRAM explanation : The 5128 chip present in [OBD0](/cars/electronics/obd0) [ECU](/cars/electronics/ecu)s is a 2K byte(?) [SRAM](/cars/electronics/sram) chip. It is mapped into memory space at 1XXX hex - any external communication(MOVX A,@DPTR) with 1000 <= DPTR < `2000h` will address the [SRAM](/cars/electronics/sram) chip. see [OBD0 PM6 PM7 RAM_Locations](/cars/electronics/obd0pm6pm7ram-locations) for some documentation on what is stored in XRAM.
