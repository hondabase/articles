# Clamping Diode

This is a diode to prevent a spike from an engine component (ike a solenoid) from damaging circuitry. In most cases a 3 amp clamping diode is more than adaquate. The following parts have been used successfully:

- 1n4001
- 1n4002
- 1n4004
