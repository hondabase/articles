# API

Application Programming Interface An [API](/cars/electronics/api) is a set of functions (usually present in a DLL or other code library) that you can call in order to perform taks. Some common [API](/cars/electronics/api)s include the Microsoft Win32 [API](/cars/electronics/api), the X windows [API](/cars/electronics/api), and the [Crome Script](/cars/electronics/crome-script) [API](/cars/electronics/api).
