# Injector Test Circuit

The Injector Test Circuit is present in [USDM](/cars/electronics/usdm) [ECU](/cars/electronics/ecu)s to determine whether injectors are connected to the [ECU](/cars/electronics/ecu). Responsible for a Code 16.
