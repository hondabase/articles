# Dual Tables

Dual tables simply refers to there being two sets of fuel/timing information in a single [ROM](/cars/electronics/rom). [VTEC](/cars/electronics/vtec) is an example of this - there is one set of tuning information for the [Low Cam](/cars/electronics/low-cam) and one for the [High Cam](/cars/electronics/high-cam). Additionally, the [ECUNitrous Control](/cars/electronics/ecu-nitrous-control) plans to include two sets of tuning information so that the [ECU](/cars/electronics/ecu) can tune for nitrous. See also: [Dual Roms](/cars/electronics/dual-roms)
