# OBD1 Conversion Formulae

- [OBD1_8bit Low Cam RPM](/cars/electronics/obd1-8bit-low-cam-rpm) - eight bit values used for low-cam tables and VTEC crossover points
- [OBD1_8bit High Cam RPM](/cars/electronics/obd1-8bit-high-cam-rpm) - eight bit values used for high-cam tables
- [OBD1_16bit RPM](/cars/electronics/obd1-16bit-rpm) - 16-bit [RPM](/cars/electronics/rpm)s used throughout the code (including revlimits, etc)
- [OBD1_8bit MBar](/cars/electronics/obd1-8bit-m-bar) - Absolute pressure scale used in all tables
- [OBD1_8bit Fuel](/cars/electronics/obd1-8bit-fuel) - Fuel values used in all tables
- [OBD1_8bit Advance](/cars/electronics/obd1-8bit-advance) - Ignition advance used in all tables
