# Oki80C154

[MCU](/cars/electronics/mcu) used in [OBD0 MPFI](/cars/electronics/obd0mpfi) [ECU](/cars/electronics/ecu)s that has zero internal code memory. requires the use of an external [EPROM](/cars/electronics/eprom) It is mostly 8051 compatible, and features a third timer like many other replacements. The main difference from a pure 8051 revolves around how the [A5](/cars/electronics/a5) instruction is handled.
