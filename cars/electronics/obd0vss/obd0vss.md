# OBD0VSS

Author: George Date: 12-03-02 22:36 [Vehicle Speed Sensor](/cars/electronics/vehicle-speed-sensor) ([VSS](/cars/electronics/vss)), stored in [RAM](/cars/electronics/ram) location 06Ch does appear to be linear (to my surprise!), but can only read up to ~157kph. Using Excel the formula is approximately: Speed (kph) = 0.6229x - 2.1385
