# RTP

Real Time Programmability The ability to change the program/tuning of an [ECU](/cars/electronics/ecu) while the [ECU](/cars/electronics/ecu) is running the motor. Currently, only the [Zdyne](http://www.zdyne.com) SECU Gold and the [AEM](http://www.aempower.com) [EMS Standalone](http://www.aempower.com/product_ems.asp) support this. Achieving real time programmability with a stock [ECU](/cars/electronics/ecu) is a significant goal for the PGMFI project.
