# Speed Limiter

A speed limiter limits the top speed of a car. In almost all hondas, a [Speed Limiter](/cars/electronics/speed-limiter) will be implemented as a piece of code that reads the [Vehicle Speed Sensor](/cars/electronics/vehicle-speed-sensor) and cuts fuel above a certain speed. [JDM](/cars/electronics/jdm) [ECU](/cars/electronics/ecu)s are famous for having speed limiters. If you want to go faster than 180kph without reprogramming [ECU](/cars/electronics/ecu) you can use Hondainfo's Speed Limit Defencer device. [Hondainfo Speed Limit Defencer](http://cvs.sourceforge.net/viewcvs.py/pgmfi/Speed%20Limit%20Defencer/)
