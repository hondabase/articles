# Peak And Hold Injectors

[Peak And Hold Injectors](/cars/electronics/peak-and-hold-injectors) aka [Low Impedance Injectors](/cars/electronics/low-impedance-injectors). Impedance typically ~2-5ohms. At 12volts, this is a draw of ~6-2.4 amps The high current will cause the injectors to run hotter. The peak current opens the injector "crisply" and decisively. Then a lower "Hold" current keeps the injector open. To run these injectors with the honda ECU, you may need to use an inline [Resistor Box](/cars/electronics/resistor-box). [OBD0](/cars/electronics/obd0) already has a resistor box wired in. [OBD1](/cars/electronics/obd1) does not have a [Resistor Box](/cars/electronics/resistor-box). [OBD2](/cars/electronics/obd2) does not have a [Resistor Box](/cars/electronics/resistor-box).
