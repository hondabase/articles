# ECU Nitrous Control

ppl are working on a [ROM](/cars/electronics/rom) where the [ECU](/cars/electronics/ecu) activates the nitrous solenoid instead of the usual [WOT](/cars/electronics/wot) switch. The main advantage of this is that the [ECU](/cars/electronics/ecu) knows when nitrous is active, and can therefore switch to a different set of fuel/timing maps. Dry kit + 450cc injectors + [ECUNitrous Control](/cars/electronics/ecu-nitrous-control) w/ [Dual Tables](/cars/electronics/dual-tables) = FINE TUNED wet kit!
