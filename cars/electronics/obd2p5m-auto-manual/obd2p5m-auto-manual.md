# OBD2P5M Auto Manual

This is a very simple conversion. Move the SMD 0 ohm resistor from location RP13 on the bottom of the [PCB](/cars/electronics/pcb) to location RP14 on the top of the [PCB](/cars/electronics/pcb). The 0 ohm SMD resistor is identified by code 000 etched on it.
