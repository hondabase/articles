# Crome ROM Editor

Crome is ROMEditor that John Cui wrote. Its main feature that differentiates itself from other ROMEditors is that it supports the use of [Java Script](/cars/electronics/java-script) to control it and implement new features. The [Crome Script](/cars/electronics/crome-script) [API](/cars/electronics/api) allows you to extend the capabilites of the editor with relative ease. People have already written many scripts for Crome, most of which can be found on the PGMFI [Source Forge](/cars/electronics/source-forge) page. The [Crome FAQ](/cars/electronics/crome-faq) is a resource designed to get you started with Crome. More information can be found at the [Crome forum](/pgmfi/forum/forum.php?id=35) or at [http://www.tunewithcrome.com/](http://www.tunewithcrome.com/).
