# 1 Wire VTEC

The 1-wire VTEC codebase is a [ROM](/cars/electronics/rom) for the [OBD0 MPFI](/cars/electronics/obd0mpfi) [ECU](/cars/electronics/ecu)s that allows the [Automatic Transmission Lockup Solenoid](/cars/electronics/automatic-transmission-lockup-solenoid) to be used to control VTEC engagement. Jason Parker came up with the [Hardware For One Wire VTEC](/cars/electronics/hardware-for-one-wire-vtec) changes. As of Oct 2 2003 it is functional but needs some help and further testing. Dave Blundell has several cars running around on it. If you want to contribute/work/see it, check out the ***1w_vtec*** code module from [Source Forge](/cars/electronics/source-forge)
