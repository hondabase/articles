# OBD1 16bit RPM

If ''x'' is a 16-bit [RPM](/cars/electronics/rpm) value, then:

- [RPM](/cars/electronics/rpm)(''x'') = 1875000/''x''

16-bit values are invariably stored in [Little Endian](/cars/electronics/little-endian) format on all [OBD1](/cars/electronics/obd1) [ECU](/cars/electronics/ecu)s.
