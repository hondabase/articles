# P51 ECU Technical Reference

The P51 ECU is an OBD1 engine control unit utilized in the 1992–1993 JDM Honda Ascot Innova 2.3L Si-Z, equipped with the H23A DOHC non-VTEC engine.

## ECU Overview

| Feature | Specification |
| :--- | :--- |
| **OBD Generation** | OBD1 |
| **Market** | JDM (Japan Domestic Market) |
| **Vehicle** | Ascot Innova (CC4) |
| **Engine** | H23A (DOHC Non-VTEC) |

> [!NOTE]
> The P51 is a non-VTEC ECU. It is not natively compatible with VTEC-equipped H-series engines without hardware modification and software remapping.

## Compatibility and Usage

The P51 is specific to the H23A non-VTEC platform. When performing diagnostics or tuning, ensure the following:

* **Sensor Compatibility:** Verify that the engine harness matches the OBD1 pinout standard for the CC4 chassis.
* **Diagnostics:** Use the standard Honda OBD1 diagnostic procedure to retrieve error codes.
