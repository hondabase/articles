# PK3 ECU Technical Reference

The PK3 ECU was utilized in the 1990–1991 Honda Prelude equipped with the B20A DOHC non-VTEC engine.

> [!NOTE]
> This ECU utilizes an electronic advance ignition system.

## Specifications

| Attribute | Detail |
| :--- | :--- |
| **Vehicle** | 1990–1991 Honda Prelude |
| **Engine** | B20A (DOHC Non-VTEC) |
| **OBD Generation** | OBD0 |
| **Ignition Type** | Electronic Advance |

## System Overview

The PK3 is a specialized OBD0-era ECU designed for the B20A engine platform. Unlike later VTEC-enabled ECUs, the PK3 is optimized for the specific fuel and ignition mapping requirements of the DOHC non-VTEC B-series engine found in the BA chassis Prelude.

> [!IMPORTANT]
> Ensure all sensor grounds are verified when troubleshooting ignition timing or fuel delivery issues on this unit, as OBD0 systems are highly sensitive to ground offsets.
