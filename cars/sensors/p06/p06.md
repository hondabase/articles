# P06 ECU Technical Reference

The P06 is an OBD1 engine control unit (ECU) utilized in 1992–1995 Honda Civic DX models equipped with the D15B SOHC non-VTEC engine.

## ECU Overview

The P06 serves as the standard factory ECU for non-VTEC D-series engines during the OBD1 generation.

![P06 ECU Circuit Board](P06top.jpg)
*Internal view of the P06 OBD1 ECU circuit board.*

## Technical Specifications

| Feature | Specification |
| :--- | :--- |
| **OBD Generation** | OBD1 |
| **Engine Application** | D15B (SOHC Non-VTEC) |
| **Chassis Application** | 1992–1995 Civic (EG) |
| **Transmission** | Manual / Automatic variants exist |

> [!NOTE]
> The P06 is a common platform for ECU modification and conversion to support VTEC operation (often referred to as a "P06 to P28 conversion") due to its shared PCB architecture with VTEC-enabled units.
