# P73 OBD2 ECU Reference (1996–2000 Integra Type-R)

The P73 ECU is the factory engine control unit for the 1996–2000 Acura Integra Type-R (DC2). This unit utilizes the OBD2 communication protocol and is found in both JDM and USDM configurations.
