# A143 Transistor Specifications and Replacements

The A143 is a Silicon PNP Darlington transistor commonly utilized in Honda ECU circuitry.

## Technical Specifications
*   **Type:** Silicon PNP Darlington Transistor
*   **Base Resistor (Rb):** 4.7k Ohm
*   **Emitter-Base Resistor (Rbe):** 10k Ohm
*   **Collector-Emitter Voltage (Vceo):** -50V
*   **Collector Current (Ic):** 0.1A
*   **Power Dissipation (Pc):** 0.3W

## Component Replacements

> [!IMPORTANT]
> Do not substitute the A143 with an NTE126, as it is not a compatible replacement.

The following components are verified replacements for the A143:

*   **Panasonic:** NTE2362 (Digi-Key Part Number: 2SB1030ARACT-ND)
*   **Panasonic:** UNR411F
*   **Toshiba:** 2SA1654

## Custom Implementation
If a direct replacement is unavailable, a high-speed PNP switching transistor (such as a 2N3906) can be used to construct an equivalent circuit.

> [!CAUTION]
> Building a custom transistor circuit requires precise component placement. Ensure the 4.7k Ohm resistor is in series with the base, followed by the 10k Ohm resistor in parallel, as per standard Darlington configuration. The user assumes all responsibility for potential ECU damage resulting from improper assembly.
