# P71 ECU Technical Reference

The P71 is an OBD1 engine control unit (ECU) utilized in 1992–1995 Acura Integra models equipped with the D16A non-VTEC engine.

## ECU Variants

The P71 ECU was produced in both manual and automatic transmission configurations.

> [!NOTE]
> Ensure the ECU variant matches your vehicle's transmission type (manual vs. automatic) to avoid transmission control errors and improper shift logic.

## Technical Specifications

| Feature | Specification |
| :--- | :--- |
| **OBD Generation** | OBD1 |
| **Engine Application** | D16A (Non-VTEC) |
| **Vehicle Application** | 1992–1995 Acura Integra |
| **VTEC Support** | No |

> [!IMPORTANT]
> This ECU does not contain the necessary hardware components (solenoid drivers) to operate VTEC systems. Attempting to run a VTEC engine with this ECU will result in the VTEC solenoid failing to engage.
