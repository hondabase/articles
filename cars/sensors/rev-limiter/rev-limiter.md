# Rev Limiter

A rev limiter limits how fast a motor can spin. Most honda [ECU](/cars/ecu/ecu)s implement a [Fuel Cut](/cars/rom/fuel-cut) [Rev Limiter](/cars/sensors/rev-limiter).
