# P29 OBD1 ECU Technical Reference

The P29 is an OBD1 engine control unit (ECU) utilized primarily in Japanese Domestic Market (JDM) Honda Domani and Civic 25th Anniversary models. It is specifically calibrated for the DOHC ZC engine.

## Overview
The P29 ECU is a standard OBD1-spec unit. It shares the same physical architecture as other common OBD1 ECUs (such as the P30 or P06) but features unique fuel and ignition maps tailored for the DOHC ZC powerplant.

> [!NOTE]
> While the P29 is physically compatible with other OBD1 Honda chassis, ensure the engine harness and sensor configuration match the DOHC ZC requirements before installation.
