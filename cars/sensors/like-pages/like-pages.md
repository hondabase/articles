# Like Pages Reference

Like Pages are a navigational feature used to identify and group documentation entries that share an initial or final word. This functionality assists in locating related technical content by surfacing pages with similar naming conventions or subject matter.

> [!NOTE]
> This feature is primarily used for cross-referencing documentation modules that share common terminology or system classifications.
