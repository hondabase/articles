# P70 OBD1 ECU Technical Reference

The P70 is an OBD1 engine control unit (ECU) utilized in 1992–1995 Honda Domani vehicles equipped with the SOHC VTEC ZC engine.

> [!NOTE]
> This ECU variant does not support or monitor the oil pressure sensor input.

## Technical Specifications

| Feature | Specification |
| :--- | :--- |
| **OBD Generation** | OBD1 |
| **Engine Application** | SOHC VTEC ZC |
| **Market** | JDM (Domani) |
| **Oil Pressure Sensor** | Not supported |
