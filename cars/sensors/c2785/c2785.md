# C2785 NPN Switching Transistor

The C2785 is a high-speed NPN switching transistor commonly utilized in Honda ECU circuitry. It is functionally interchangeable with the C144 transistor.

## Replacement Components

The following components are verified direct replacements for the C2785:

| Manufacturer | Part Number |
| :--- | :--- |
| NTE | NTE2361 |
| Panasonic | 2SD1423ARACT-ND |
| Toshiba | C2458Y |

> [!TIP]
> Refer to the [NTE2361 Datasheet](http://www.nteinc.com/specs/2300to2399/pdf/nte2361.pdf) for detailed electrical characteristics and pinout configuration.
