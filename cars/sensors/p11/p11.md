# P11 ECU Hardware Overview (92-95 Honda Prelude)

The P11 ECU is an OBD1 engine control unit utilized in the 1992–1995 Honda Prelude 2.0i (BB3 chassis) equipped with the F20A4 non-VTEC engine.

## Technical Specifications

*   **Chassis:** BB3
*   **Engine:** F20A4 (Non-VTEC)
*   **OBD Standard:** OBD1
*   **Market:** European/General

> [!NOTE]
> The P11 is a non-VTEC ECU. It lacks the necessary hardware components (transistors and solenoids) to drive a VTEC solenoid or monitor a VTEC oil pressure switch.

## Hardware Configuration

The P11 utilizes a standard OBD1 architecture common to the 92-95 Honda lineup. 

### ECU Pinout Reference
The P11 follows the standard Honda OBD1 26-pin, 16-pin, and 22-pin connector layout.

## Compatibility and Tuning

The P11 is a non-VTEC unit. If performing a VTEC engine swap, this ECU will require hardware modification (adding VTEC components) and a software reflash to support VTEC engagement, or it should be replaced with a VTEC-compatible unit (e.g., P13 or P72).

> [!CAUTION]
> Attempting to run a VTEC engine on a stock P11 ECU without modification will result in the engine operating without VTEC engagement, potentially causing performance loss and incorrect fuel/ignition mapping for the VTEC profile.
