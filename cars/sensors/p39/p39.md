# P39 OBD1 ECU Technical Reference

The P39 is an OBD1 engine control unit (ECU) utilized in 1992–1995 JDM Honda Prelude (BB4) chassis vehicles equipped with the F22B DOHC non-VTEC engine.

## ECU Overview

The P39 is typically found in automatic transmission configurations. Hardware modifications are required to convert the unit for manual transmission operation.

## Transmission Conversion

To convert an automatic P39 ECU for use with a manual transmission, the following hardware modification is required:

> [!IMPORTANT]
> Moving the resistor from the **RP11** position to the **RP12** position is necessary to signal the ECU to operate in manual transmission mode.
