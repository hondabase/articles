# UberData Tuning Software Overview

UberData is a legacy software platform developed for the modification and remapping of Honda OBD1 engine control units (ECUs). It served as one of the foundational tools in the early development of open-source Honda ECU tuning.

## Overview
UberData allows users to modify fuel and ignition maps, adjust rev limiters, and disable specific sensor checks within the ECU's ROM. It utilizes a graphical interface to translate hexadecimal data into editable tables.

> [!NOTE]
> UberData is considered legacy software. Most modern tuning workflows have transitioned to more current platforms that offer improved stability, real-time tuning capabilities, and broader hardware support.

## Key Features
* **Map Editing:** Provides basic 2D and 3D visualization for fuel and ignition tables.
* **Parameter Adjustment:** Allows for the modification of VTEC engagement points, rev limiters, and speed limiters.
* **Datalogging:** Supports basic datalogging functionality to monitor engine parameters during operation.
