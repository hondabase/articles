# API

Application Programming Interface An [API](/cars/diagnostics/api) is a set of functions (usually present in a DLL or other code library) that you can call in order to perform taks. Some common [API](/cars/diagnostics/api)s include the Microsoft Win32 [API](/cars/diagnostics/api), the X windows [API](/cars/diagnostics/api), and the [Crome Script](/cars/rom/crome-script) [API](/cars/diagnostics/api).
