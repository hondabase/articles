# 1991 Honda PM6 ECU Reference

The PM6 ECU is a common engine control unit found in 1988–1991 Honda Civic and CRX models equipped with the D16A6 engine. This article serves as a technical reference for configuration and diagnostic procedures.

## Idle Configuration
The target idle speed for the PM6 ECU is managed via specific memory addresses within the ECU firmware. 

> [!TIP]
> The procedure for adjusting the target idle on a PM6 ECU is functionally identical to the process used for the PM7 ECU. Refer to the [91 PM7 Target Idle](/cars/tuning/91pm7-target-idle) documentation for step-by-step instructions.

## Diagnostics
The PM6 ECU features an onboard diagnostic LED located on the ECU casing to assist in identifying system faults.

*   **Diagnostic LED:** The LED flashes to indicate stored Diagnostic Trouble Codes (DTCs).
*   **Troubleshooting:** For a comprehensive guide on interpreting the blinking patterns and identifying specific engine faults, refer to the [91 PM6 ECU LED CEL](/cars/diagnostics/91pm6-ecu-led-cel) documentation.
