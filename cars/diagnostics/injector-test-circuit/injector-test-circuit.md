# Injector Test Circuit

The Injector Test Circuit is present in [USDM](/cars/sensors/usdm) [ECU](/cars/ecu/ecu)s to determine whether injectors are connected to the [ECU](/cars/ecu/ecu). Responsible for a Code 16.
