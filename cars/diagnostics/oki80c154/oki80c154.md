# Oki80C154

[MCU](/cars/rom/mcu) used in [OBD0 MPFI](/cars/diagnostics/obd0mpfi) [ECU](/cars/ecu/ecu)s that has zero internal code memory. requires the use of an external [EPROM](/cars/rom/eprom) It is mostly 8051 compatible, and features a third timer like many other replacements. The main difference from a pure 8051 revolves around how the [A5](/cars/diagnostics/a5) instruction is handled.
