# Uberdata and ERM Tuning Editors

Uberdata and ERM are legacy ECU tuning editors. These platforms were among the first publicly available tools to provide native support for boost-based fuel and ignition mapping in Honda OBD-series engine management systems.

## Overview
These editors allow for the modification of ROM files to accommodate forced induction setups, including:
*   Custom fuel maps for boost pressure.
*   Ignition timing retard under load.
*   Sensor scaling for aftermarket MAP sensors.

> [!NOTE]
> These tools are considered legacy software. For modern tuning applications, it is recommended to use current-generation platforms that offer improved stability, real-time tuning capabilities, and wider hardware support.
