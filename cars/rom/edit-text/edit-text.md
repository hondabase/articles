# Contributing to Hondabase Documentation

All technical pages within the Hondabase documentation—excluding search results—feature an **Edit Text** link located at the bottom of the page. This functionality allows users to submit updates, corrections, or additional technical data directly to the repository.

> [!TIP]
> To contribute, navigate to the bottom of any technical article and select the **Edit Text** link to access the source content for that specific page.

For further information regarding the underlying documentation framework and formatting standards, refer to the [Documentation Reference](/cars/reference/php-wiki-documentation).
