# P61 ECU Technical Reference

The P61 is an OBD1 engine control unit (ECU) utilized in 1992–1993 USDM Acura Integra GSR models equipped with the B17A DOHC VTEC engine. It features the m66207-215 microcontroller.

## Hardware Overview

The P61 is notable for its specific board layout designed to support the B17A engine management requirements.

### External ROM Board
The P61 utilizes an external ROM board for fuel and ignition mapping.

## Technical Specifications

> [!IMPORTANT]
> The P61 ECU is specific to the B17A engine. While physically similar to other OBD1 ECUs, internal component values and logic are tuned specifically for the B17A's intake and VTEC engagement parameters.

| Feature | Specification |
| :--- | :--- |
| **OBD Standard** | OBD1 |
| **Market** | USDM |
| **Engine** | B17A (DOHC VTEC) |
| **Microcontroller** | m66207-215 |
| **Chassis** | DA (1992-1993 Integra) |
