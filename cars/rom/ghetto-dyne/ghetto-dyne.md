# Ghetto Dyne Tuning Platform

Ghetto Dyne was an early ECU tuning solution for Honda engine management systems. 

> [!NOTE]
> The Ghetto Dyne platform is considered legacy software. For current tuning applications, [Turbo Edit](/cars/rom/turbo-edit) provides comprehensive functionality and is actively maintained.

## Overview
Ghetto Dyne served as a foundational tool for early OBD-based Honda ECU modification. Users seeking historical documentation or technical archives regarding the original implementation can refer to archived resources:

* [Ghetto Dyne Archived Documentation](http://web.archive.org/web/20110128180504/http://ghettodyne.com/)

## Migration to Turbo Edit
Turbo Edit has superseded the Ghetto Dyne platform. It offers full feature parity with the original Ghetto Dyne software, including:

* Enhanced ROM editing capabilities.
* Improved sensor data handling.
* Active development and community support.

For further technical discussion or support regarding legacy tuning methods, refer to the [GhettoDyno forum](/pgmfi/forum/forum.php?id=11).
