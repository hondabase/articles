# NEC 2SD1780 Transistor Specifications

The NEC 2SD1780 is a silicon NPN power transistor frequently utilized in Honda ECU hardware circuits.

## Technical Specifications

The following table outlines the primary electrical characteristics and pin configuration for the 2SD1780.

| Parameter | Specification |
| :--- | :--- |
| **Type** | NPN |
| **Collector Current (Ic)** | 2 A |
| **Collector-Emitter Voltage (Vceo)** | 50 V |
| **DC Current Gain (hFE)** | 800 – 3200 |
| **Pin Configuration** | E-C-B (1-2-3) |

> [!NOTE]
> Pin configuration is defined as Pin 1: Emitter, Pin 2: Collector, Pin 3: Base.

## Cross-Reference

If the original NEC 2SD1780 is unavailable, the Panasonic 2SD2178 is a suitable functional replacement. This component is commonly available through major electronics distributors such as Digi-Key.

> [!IMPORTANT]
> Always verify the datasheet of any replacement component against the specific circuit requirements of the ECU board before installation to ensure compatibility with existing board traces and thermal dissipation needs.
