# User Preferences Configuration

Hondabase allows users to customize site behavior and display settings. These preferences are managed locally via browser cookies.

> [!IMPORTANT]
> Browser cookies must be enabled in your web browser settings for these preferences to be saved and applied across sessions. If cookies are disabled or cleared, all settings will revert to their default values.

## Managing Preferences

To update your user preferences, navigate to the settings interface. Once configured, the site will automatically reference the stored cookie data to adjust the interface according to your saved profile.

### Available Settings
*   **Display Mode:** Toggle between light and dark themes.
*   **Unit Conversion:** Select preferred measurement systems (Metric vs. Imperial) for technical specifications.
*   **Data View:** Configure default sorting and filtering for ECU pinout tables and error code databases.

> [!TIP]
> If you experience unexpected behavior or display issues, ensure your browser is not blocking cookies for this domain. Clearing your browser cache and cookies is the recommended troubleshooting step if settings fail to persist.
