# Desoldering Braid

Desoldering braid is a fine woven generally copper wire used to remove solder by using the "wicking" action that draws liquids into a fabric-like material. (Much like a paper towel soaks up water) Desoldering braid is cheap, relatively effective and can be found even at shitty electronics stores like [Rat Shack](/cars/sensors/rat-shack).
