# RM11 Resistor Network Replacement

In some Honda ECU VTEC conversions, the **RM11** resistor network component may need to be bypassed or replaced with individual discrete resistors to allow for proper ECU configuration or VTEC solenoid control.

---

## Replacement Specification
The RM11 component can be replaced by installing three discrete resistors in the following configuration:

*   **Pin 1 to Pin 2:** 10k $\Omega$ Resistor
*   **Pin 2 to Pin 3:** 51k $\Omega$ Resistor
*   **Pin 3 to Pin 8:** 12k $\Omega$ Resistor

---

## Technical Considerations
*   **Diagnostics:** This replacement allows for normal ECU operation and proper VTEC solenoid feedback functionality.
*   **Code 21 (MIL):** Proper installation of these resistors ensures the ECU correctly reads the VTEC solenoid status and avoids triggering Code 21.
*   **Automatic ECUs:** This replacement may not function correctly for automatic transmission ECUs if they require feedback from the automatic transmission lockup solenoid. A true RM11 component or additional load-simulation resistors may be required for automatic applications.

*Always verify your board layout and component markings before soldering. If you suspect an RM11 has failed, carefully measure the existing resistance values before attempting a replacement to ensure they match the requirements for your specific ECU part number.*
