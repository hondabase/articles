# P07 OBD1 ECU Reference (Civic VX)

The **P07** is a unique OBD1 ECU equipped in the 1992–1995 Civic VX (D15Z1 engine). It is distinguished by its dual-processor architecture, designed to manage the Civic VX's lean-burn combustion system.

---

## Technical Overview
*   **Dual-Processor Architecture:** Features two `66207` processors, one of which is dedicated to managing the specialized wideband-like lean-burn O2 sensor system.

---

## Technical Data

### RAM Map
| Location | Bytes | Description | Notes |
| :---: | :---: | :--- | :--- |
| **00CC** | 1 | Vehicle Speed | (km/h) |
| **00D9** | 1 | ECT Sensor | 0V–5V (`0x00`–`0xFF`) |

### ROM Map
| Location | Bytes | Description | Notes |
| :---: | :---: | :--- | :--- |
| **1292** | 1 | VTEC Coolant Temp Check | `0xD8` enables, `0xFF` disables |
| **1997** | 1 | Speed Limiter | B7 = 183 km/h (114mph); FE = 254 km/h (158mph) |
| **1998** | 2 | Speed Limiter Routine | Bypass via NOP instructions (`00 00`) |
| **2BA3** | 2 | Checksum Jump | Change `JEQ` to `SJ` to disable |

---

*Note: The P07 ECU architecture is significantly different from standard VTEC ECUs. ROM modifications require specific care to maintain the integrity of the lean-burn control algorithms.*
