# OBD0CN3

[OBD0](/cars/rom/obd0) [ECU](/cars/ecu/ecu)s generally have a 5 pin serial port labelled ***"`CN3`"***This is the same pinout and has the same function more or less as [OBD1 CN2](/cars/wiring/obd1cn2)
