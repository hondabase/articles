# P08 OBD1 ECU Reference

The **P08** is an OBD1 ECU commonly found in 1992–1995 Civic/CRX VTi/VXi models powered by the D15B SOHC VTEC engine.

---

## Technical Overview
*   **Engine:** D15B (SOHC VTEC)
*   **O2 Sensor:** 1-wire unheated configuration.
*   **Compatibility:** Electrically similar to the P28 ECU, making it a viable alternative for many D-series SOHC VTEC applications.

*While similar to the USDM P28, ensure you verify O2 sensor heating requirements when running P08 code on different harness setups, as missing the 1-wire O2 sensor can trigger diagnostic codes.*
