# Pocket Programmer2

O [Pocket Programmer 2](https://www.xtronics.com/memory/EPROM.htm) tem um preço listado de $149,95 em 9 de outubro de 2003. O seu preço não tem oscilado muito. Suporta uma grande variedade de chips e possui um software decente para Windows e DOS. Consegue gravar [MCU](/cars/rom/mcu)s e todo o tipo de chips estranhos através de uma variedade de adaptadores adicionais. Este é um gravador de ROM ([Rom Burner](/cars/ecu/rom-burner)) muito comum, pois era vendido em conjunto com a [Hondata](http://www.hondata.com). Na minha humilde opinião, o Pocket Programmer 2 não é um mau negócio, particularmente para quem procura um produto polido. O suporte técnico da Xtronics é bastante razoável, eles mantêm o software atualizado, mas já vi muitos destes gravadores serem muito, muito, muito peculiares. Nunca consegui gravar uma [ROM](/cars/rom/rom) com o meu antigo ThinkPad T20, mas funcionou perfeitamente com o meu ThinkPad T21 atual. Não é um mau gravador para o preço, mas depois de usar o [Super Pro Z](/cars/diagnostics/super-pro-z) eu nunca mais compraria um. -Dave
