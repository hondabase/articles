# Adicionar Páginas

# Esta wiki não é de facto editável. É mantida por Sanimalp apenas para referência. Ele pode ser contactado via PM em Honda-Tech.com, utilizador sanimalp, ou através de hotmail.com usando sanimalp com a notação @.

Para adicionar uma nova página à Wiki, tudo o que tem de fazer é escolher um título significativo, colocar a primeira letra de cada palavra em maiúscula e Juntá-lasDestaForma. A Wiki reconhece automaticamente isto como uma hiperligação. Alternativamente, pode colocar qualquer coisa em [parênteses retos](/cars/reference/square-brackets). Depois pode clicar no Ponto de Interrogação no fim da sua nova hiperligação, e a Wiki abrirá uma janela para criar a nova página. Se deseja adicionar documentos com formatação complexa à Wiki, poderá ser preferível fornecer um URL para o mesmo em vez de tentar adicionar o texto do documento aqui, como por exemplo: página inicial do projeto [Php Wiki](/cars/sensors/php-wiki): [http://web.archive.org/web/20200322071116/http://web.archive.org/web/20200322071116/http://phpwiki.sourceforge.net/](http://web.archive.org/web/20200322071116/http://web.archive.org/web/20200322071116/http://phpwiki.sourceforge.net/) A Wiki não suporta tags HTML (ver [Regras de Formatação de Texto](/cars/diagnostics/text-formatting-rules)). Elas serão apenas renderizadas como texto. A Wiki foi concebida para ser tão simples quanto possível para incentivar a utilização. Note que existe uma funcionalidade que o administrador do seu sistema pode ativar para permitir HTML incorporado, mas existem riscos de segurança envolvidos. ---
