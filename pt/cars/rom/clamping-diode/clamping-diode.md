# Díodo de Clamping

Este é um díodo para evitar que um pico de tensão de um componente do motor (como um solenoide) danifique os circuitos. Na maioria dos casos, um díodo de clamping de 3 amperes é mais do que suficiente. Os seguintes componentes foram utilizados com sucesso:

- 1n4001
- 1n4002
- 1n4004
