# Crome ROM Editor

Crome is a ROM editor that distinguishes itself from competing tools through integrated JavaScript scripting support. The Crome Script API enables developers to extend the editor's capabilities with custom tuning functions and data processing workflows.

## Key Features

- **JavaScript-Based Scripting:** Control editor behavior and implement custom functionality using Crome Script API
- **Multi-OBD Support:** Compatible with OBD0, OBD1, and OBD2 calibrations
- **Extensible Architecture:** Large community-contributed script library available

- **Crome Script API:** See [Crome Script](/cars/rom/crome-script) for JavaScript API reference
- **Frequently Asked Questions:** Refer to [Crome FAQ](/cars/rom/crome-faq) for common setup and usage topics
- **Community Scripts:** Extensive script library available via SourceForge repository
