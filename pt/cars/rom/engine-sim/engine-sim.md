# Engine Simulator

The Engine Simulator is a device designed to simulate running engine signals for [ECU](/cars/ecu/ecu) development and bench testing applications.

## Overview

An engine simulator generates realistic sensor outputs and load patterns to allow ECU firmware development and validation without a running engine. This is critical for safe, controlled testing during ROM tuning and diagnostic work.
