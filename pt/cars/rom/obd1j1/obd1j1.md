# OBD1J1

O jumper `J1` nas [ECU](/cars/ecu/ecu)s [OBD1](/cars/wiring/obd1) controla o estado de [_EA](/cars/rom/ea). É utilizado para alterar se o [MCU](/cars/rom/mcu) utiliza uma [ROM](/cars/rom/rom) externa ou o programa interno.
