# Aviso Sobre a Adição de um Sensor de Detonação

A frequência do sensor de detonação e, consequentemente, o seu funcionamento, são afetados pelo diâmetro (bore) do cilindro. Pelo que percebo, a placa de detonação de `cada` [ECU](/cars/ecu/ecu) também está sintonizada com as características específicas do motor, para filtrar o ruído do motor face à detonação... usar uma placa de detonação com um motor diferente daquele para o qual foi desenhada pode não ser muito eficaz...
