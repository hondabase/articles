# Apagamento por UV (UV-Erase)

Muitos chips têm uma janela de vidro transparente na parte superior. Se os expuser a ***UV de onda curta*** (ou seja, o tipo que causa cancro instantâneo) a partir de uma lâmpada esterilizadora ou de uma lâmpada de apagamento por UV durante tempo suficiente, a informação guardada no chip é apagada e este pode ser reutilizado.

**Nota:** as "luzes negras" que os hippies gostam de usar nos seus quartos não são fiáveis para apagar chips. Nota #2: a luz solar não é fiável para apagar chips. Nota #3: este [link](http://www.qsl.net/iz7ath/web/02_brew/15_lab/05_canc/english/pag01_eng.htm) explica como construir um apagador.
