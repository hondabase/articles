# 66k Resources

### Documentação

- [66k Assembler Docs](/cars/diagnostics/66k-assembler-docs) - documentação de fábrica sobre os processadores Oki 66xxx usados na maioria das implementações [OBD1](/cars/wiring/obd1) e [OBD2](/cars/wiring/obd2)
- Operações 66k - uma lista das operações comuns de assembler 66xxx e como são usadas na maioria das implementações PGM-FI

### Utilitários

- [ASM662](/cars/diagnostics/asm662) - assembler/disassembler de código aberto para os processadores Oki Nx8/200 66k por Andy Sloane
- [66507 DASM](/pgmfi/forum/topic.php?id=1765) - disassembler para o processador [OBD2](/cars/wiring/obd2) Oki `66507` por Pelegri Didier
- [assembly-Oki66k.xml](/pgmfi/forum/topic.php?id=1769) - ficheiro de definição [Edit Mode](http://jedit.org/users-guide/installing-modes.html) do jEdit para assembly 66k por Dave Johnson
- [okiedit](/pgmfi/forum/topic.php?id=1770) - editor com destaque de sintaxe Win32 para assembly 66k por John Cui
- [asm syntax for textpad 4.6](/pgmfi/forum/topic.php?id=1772) - sintaxe para textpad por Calvin Baank
