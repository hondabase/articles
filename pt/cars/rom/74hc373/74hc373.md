# 74HC373 Octal Latch

O **74HC373** é um chip latch octal. Ele [retém](/cars/rom/latch) (latch) 8 bits de dados e é normalmente utilizado ao adicionar uma ROM externa a um [MCU](/cars/rom/mcu).
