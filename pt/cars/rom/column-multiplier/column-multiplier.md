# Multiplicador de Coluna

O multiplicador de coluna é um número utilizado numa fórmula para calcular uma coluna do mapa de combustível. Este multiplicador encontra-se mesmo no fim do referido mapa de combustível. O primeiro número a seguir ao mapa de combustível é o multiplicador para a coluna 1 do mapa, o segundo número é para a coluna 2 e assim sucessivamente.
