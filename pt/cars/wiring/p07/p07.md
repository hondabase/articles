# P07

A P07 92-95 [OBD1](/cars/wiring/obd1) do Civic VX (D15Z1) é única no sentido em que possui dois processadores `66207` (um presumivelmente para a sonda lambda de banda larga/wideband O2).

**Nota:** O código 303 P07 é muito semelhante ao código 304 P28-A01, pelo menos os vetores são os mesmos!
