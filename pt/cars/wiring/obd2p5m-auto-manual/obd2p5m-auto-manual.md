# OBD2P5M Auto para Manual

Esta é uma conversão muito simples. Mova a resistência SMD de 0 ohm da localização RP13 na parte inferior da [PCB](/cars/wiring/pcb) para a localização RP14 na parte superior da [PCB](/cars/wiring/pcb). A resistência SMD de 0 ohm é identificada pelo código 000 gravado nela.
