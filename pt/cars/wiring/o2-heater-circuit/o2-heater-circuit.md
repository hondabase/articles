# Circuito de Aquecimento de O2

O circuito de aquecimento do sensor de O2 é o que diferencia os [Sensores de Oxigénio](/cars/fueling/oxygen-sensor) de 4 fios dos sensores de 1 fio. O circuito de aquecimento ajuda o sensor a manter-se na sua faixa de temperatura ideal para que a leitura seja precisa. Consulte [Sensor de O2 de 1 Fio para 4 Fios](/cars/honda/civic/eg/wiring/one-wire-to4-wire-o2-sensor) para informações sobre como adicionar circuitos a uma [ECU](/cars/ecu/ecu) para suportar o [Circuito de Aquecimento de O2](/cars/wiring/o2-heater-circuit). Consulte os [Mapas de ROM](/cars/wiring/rom-maps) para informações sobre como desativar o código 41 apresentado quando este está ausente ou avariado.
