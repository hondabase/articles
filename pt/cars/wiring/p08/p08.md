# P08 OBD1 ECU (1992–1995 Civic/CRX VTi/VXi)

The P08 is an OBD1 engine control unit designed for 1992–1995 Honda Civic and CRX models equipped with the D15B SOHC VTEC engine. It features a single-wire oxygen sensor configuration and shares similar core architecture with the P28 ECU.

> [!NOTE]
> This ECU is functionally similar to the P28 with the primary difference being the single-wire oxygen sensor design.

## Key Specifications

| **Attribute** | **Details** |

| **Engine** | D15B SOHC VTEC |
| **Years** | 1992–1995 |

| **Oxygen Sensor Type** | Single-wire |
