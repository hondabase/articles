# Source Forge

O [Source Forge](/cars/wiring/source-forge) é uma comunidade Open Source para tudo. Atualmente está a ser utilizado para o [ASM662](/cars/diagnostics/asm662), desenvolvimento da [ROM](/cars/rom/rom) P30 de Andy Sloane, a codebase [PM6 Future](/cars/diagnostics/pm6-future), a codebase [1 Wire VTEC](/cars/wiring/1-wire-vtec) e scripts para o [Crome ROMEditor](/cars/rom/crome-rom-editor). A página da PGMFI no SourceForge pode ser encontrada em [http://web.archive.org/web/20260228000421/https://sourceforge.net/projects/pgmfi/](http://web.archive.org/web/20260228000421/https://sourceforge.net/projects/pgmfi/). Se quiser disponibilizar algum do seu tempo para criar uma página web para os recursos do SourceForge ([http://web.archive.org/web/20131206065104/http://web.archive.org/web/20131206065104/http://pgmfi.sourceforge.net:80/:80/](http://web.archive.org/web/20131206065104/http://web.archive.org/web/20131206065104/http://pgmfi.sourceforge.net:80/:80/) - atualmente em branco), por favor envie um e-mail a Dave Blundell, Andy Sloane ou Calvin Baank (ou qualquer um dos outros administradores do SourceForge) se desejar envolver-se. Nota: se tentar descarregar (check out) módulos do SourceForge, preste atenção às letras maiúsculas e minúsculas! O [Source Forge](/cars/wiring/source-forge) distingue maiúsculas de minúsculas (CASE SENSITIVE)!!!
