# OBD0

"OBD0" não é um padrão oficial como o [OBD2](/cars/wiring/obd2). É mais comummente utilizado para referir as [ECU](/cars/ecu/ecu)s [MPFI](/cars/sensors/mpfi) fabricadas entre 88-91 com [distribuidores de avanço eletrónico (Electronic Advance Distributors)](/cars/sensors/electronic-advance-distributors) encontradas nos carros [OBD0 MPFI](/cars/diagnostics/obd0mpfi) desta era. Tecnicamente, todas as [ECU](/cars/ecu/ecu)s fabricadas antes de 1988 e todas as de Accord/Prelude/V6/[ECUs OBD0 VTEC](/cars/diagnostics/obd0-vtec-ec-us) desta era são também "OBD0" mas, devido às diferenças drásticas, são agrupadas separadamente e referidas como [ECU](/cars/ecu/ecu)s "pre-OBD" ou por outro termo. [ECU](/cars/ecu/ecu)s [OBD](/cars/wiring/obd)0 (88-91)

- [PG7](/cars/sensors/pg7) : 88-89 Integra (D16A1)
- [PM6](/cars/ecu/pm6) : 88-91 Civic/CRX [SOHC](/cars/sensors/sohc) Si (1.6l SOHC)
- [PM7](/cars/ecu/pm7) : 89-91 Civic Si [DOHC](/cars/sensors/dohc) ZC ([JDM](/cars/sensors/jdm) EF3)
- [PM7](/cars/ecu/pm7) : 89-91 CRX D16A8, D16A9 ( [EDM](/cars/wiring/edm) )
- [PM8](/cars/diagnostics/pm8) : 88-91 CRX HF
- [PR3](/cars/sensors/pr3) : 89-91 [JDM](/cars/sensors/jdm) B16A EF8/9
- [PR5](/cars/sensors/pr5) : 90-91 [JDM](/cars/sensors/jdm) Integra ZXi (1.6 SOHC)
- [PW0](/cars/sensors/pw0) : 89-91 [JDM](/cars/sensors/jdm)/Euro B16A EF8/9 DA6-XSi ([DOHC](/cars/sensors/dohc) VTEC)
- [PR4](/cars/sensors/pr4) : 90-91 Integra LS/GS (b18a)
- [PS9](/cars/sensors/ps9) : 88-91 Civic de 4 portas EX Auto (SOHC 1.6)

As [ECU](/cars/ecu/ecu)s PH3 e PK3 partilham quase a mesma placa de circuito impresso (PC board). - [PK2](/cars/diagnostics/pk2) : 88-91 Prelude Si motor B20A [DOHC](/cars/sensors/dohc) 140 CV
- [PH3](/cars/diagnostics/ph3) : 86-89 Accord 2.0Si B20A

Estas [ECU](/cars/ecu/ecu)s [DPFI](/cars/sensors/dpfi) também foram fabricadas durante o mesmo período de tempo: - [PM5](/cars/honda/civic/ef/tuning/pm5) : 88-91 Civic/CRX DX ([DPFI](/cars/sensors/dpfi) D15b)
- [PM9](/cars/honda/civic/ef/tuning/pm9) : 88-91 Civic modelo base (DPFID15b)
