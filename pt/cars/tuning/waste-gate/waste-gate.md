# Wastegate

Uma **wastegate** é um dispositivo que permite que os gases de escape contornem o turbocompressor. Uma vez que os turbocompressores criam pressão (boost) utilizando a energia dos gases de escape, a wastegate pode ser usada para limitar os níveis de pressão de sobrealimentação. As wastegates são geralmente configuradas para abrir a uma determinada pressão.
