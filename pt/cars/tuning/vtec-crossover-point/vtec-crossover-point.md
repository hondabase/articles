# VTEC Crossover Point

Num motor [VTEC](/cars/sensors/vtec), o **VTEC Crossover Point** é o [RPM](/cars/sensors/rpm) no qual o [solenoide VTEC](/cars/wiring/vtec-solenoid) é ativado para alternar da [Low Cam](/cars/sensors/low-cam) para a [High Cam](/cars/sensors/high-cam).
