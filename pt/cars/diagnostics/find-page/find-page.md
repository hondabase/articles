# Encontrar Página

Aqui estão alguns bons pontos de partida para navegação.

- [Como Usar a Wiki](/cars/diagnostics/how-to-use-wiki) dá-lhe uma introdução rápida à marcação Wiki.
- [Alterações Recentes](/cars/rom/recent-changes) gravadas automaticamente.
- [Mais Sobre Mecânica](/cars/rom/more-about-mechanics) de navegação e edição neste servidor.

Aqui está uma pesquisa por título. Experimente algo como ''wiki'' ou ''sandwich''. Pesquisa por Título?> Use o seguinte para uma pesquisa de texto completo. Isto demora alguns segundos. Os resultados mostrarão todas as linhas de uma determinada página que contenham uma correspondência. Pesquisa de Texto Completo?> Numa pesquisa de !FuzzyPage, os títulos de todas as páginas são examinados para encontrar aqueles com grafia semelhante ou som parecido (inglês). Páginas Fuzzy?> ---

Separe as palavras com um espaço. Todas as palavras têm de corresponder. Para excluir palavras, adicione um '-' antes delas. Exemplo: 'wiki text -php' procura todas as páginas que contêm as palavras 'wiki' ***e*** 'text', mas ***não*** contêm a palavra 'php'.
