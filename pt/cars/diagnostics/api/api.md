# API

Interface de Programação de Aplicações Uma [API](/cars/diagnostics/api) é um conjunto de funções (geralmente presentes numa DLL ou noutra biblioteca de código) que pode chamar para realizar tarefas. Algumas [API](/cars/diagnostics/api)s comuns incluem a Microsoft Win32 [API](/cars/diagnostics/api), a X windows [API](/cars/diagnostics/api) e a [Crome Script](/cars/rom/crome-script) [API](/cars/diagnostics/api).
