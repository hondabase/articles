# Ícones de Link

Os [Ícones de Link](/cars/diagnostics/link-icons) são uma nova funcionalidade opcional da [Php Wiki](/cars/sensors/php-wiki). Quando ativados, serão exibidos ícones à frente dos URLs para indicar o tipo de link. - [http://phpwiki.sourceforge.net/alpha/themes/default/images/http.png](http://phpwiki.sourceforge.net/alpha/themes/default/images/http.png) link http
- [http://phpwiki.sourceforge.net/alpha/themes/default/images/url.png](http://phpwiki.sourceforge.net/alpha/themes/default/images/url.png) link genérico de internet
- [http://phpwiki.sourceforge.net/alpha/themes/default/images/interwiki.png](http://phpwiki.sourceforge.net/alpha/themes/default/images/interwiki.png) link [Inter Wiki](/cars/rom/inter-wiki)
- [http://phpwiki.sourceforge.net/alpha/themes/default/images/https.png](http://phpwiki.sourceforge.net/alpha/themes/default/images/https.png) link https
- [http://phpwiki.sourceforge.net/alpha/themes/default/images/mailto.png](http://phpwiki.sourceforge.net/alpha/themes/default/images/mailto.png) link mailto
- [http://phpwiki.sourceforge.net/alpha/themes/default/images/ftp.png](http://phpwiki.sourceforge.net/alpha/themes/default/images/ftp.png) link ftp

;__Nota__: Alguns dos ícones padrão usam a funcionalidade de canal alfa do formato de imagem [PNG](http://www.libpng.org/pub/png/png-sitemap.html) para uma renderização suave em qualquer cor de página ou fundo texturado. Navegadores mais antigos podem não exibir todas as imagens corretamente.

# Exemplos 

Os exemplos seguintes serão exibidos com os ícones de link acima apenas se o administrador tiver ativado esta funcionalidade.

- Link [Inter Wiki](/cars/rom/inter-wiki) [Php Wiki](/cars/sensors/php-wiki):InterWiki

- Um [Inter Wiki](/cars/rom/inter-wiki) nomeado leia Why Wiki Works em c2.

## URLs

- Endereço de email mailing list PhpWiki-talk

- Link http normal [PhpWiki's Home Page](http://phpwiki.sourceforge.net/phpwiki/)

- Link http seguro [PhpWiki's Home Page](https://phpwiki.sourceforge.net/phpwiki/)

- Transferência de ficheiros [ftp://ftp.sourceforge.net/]()

''Um ícone genérico é mostrado para outros tipos de link menos comuns.'' - [news://nntp.news.com](news://nntp.news.com)

- [gopher://gopher.caltech.cmu.edu](gopher://gopher.caltech.cmu.edu)

Os [Ícones de Link](/cars/diagnostics/link-icons) apenas serão exibidos quando o protocolo URL for um daqueles para os quais a [Php Wiki](/cars/sensors/php-wiki) tem permissão de ligação. Os exemplos seguintes não mostrarão ícones e a [Php Wiki](/cars/sensors/php-wiki) também não fornecerá automaticamente links clicáveis, a menos que o administrador os adicione: - bogus://oedipus.nostradamus.com

- bolo://lgm.cheshire.org
