# Circuito de Teste dos Injetores

O Circuito de Teste dos Injetores está presente nas [ECU](/cars/ecu/ecu)s [USDM](/cars/sensors/usdm) para determinar se os injetores estão ligados à [ECU](/cars/ecu/ecu). É responsável pelo Código 16.
