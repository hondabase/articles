# Programabilidade em Tempo Real

Um sistema no qual a [EPROM](/cars/rom/eprom) da [ECU](/cars/ecu/ecu) é substituída por um sistema personalizado que é programável em tempo real. Isto permite efetuar alterações no código enquanto o motor está a funcionar. Isto é geralmente conseguido com a utilização de um portátil externo para gerir e atualizar o sistema [RTP](/cars/sensors/rtp).
