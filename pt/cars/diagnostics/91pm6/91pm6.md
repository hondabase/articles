# 1991 Honda PM6 ECU Reference

The PM6 ECU was utilized in various 1988–1991 Honda vehicles. This article serves as a central reference for diagnostic procedures and configuration settings specific to this hardware.

## Diagnostic and Configuration Procedures

The following procedures are essential for maintaining and tuning the PM6 ECU:

*   **Idle Configuration:** Refer to the [PM6 Target Idle](/cars/diagnostics/91pm6-target-idle) guide for instructions on adjusting the target idle speed. This process is functionally similar to the [PM7 Target Idle](/cars/rom/91pm7-target-idle) procedure.
*   **Diagnostic LED:** For information regarding the onboard diagnostic LED and interpreting Check Engine Light (CEL) codes, refer to the [PM6 ECU LED Diagnostics](/cars/diagnostics/91pm6-ecu-led-cel) guide.

> [!NOTE]
> Ensure the ECU is properly grounded and the main relay is functioning correctly before attempting to diagnose idle or sensor issues, as these are common points of failure in OBD0 systems.
