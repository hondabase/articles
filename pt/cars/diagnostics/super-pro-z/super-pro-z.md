# Super Pro Z

O [Xeltek](http://www.xeltek.com) [SuperProZ](http://web.archive.org/web/20200919092029/http://www.xeltek.com/product/programmer/spz.html) custa 249 $. Continua a ser um programador/gravador com interface paralela como a maioria dos outros analisados, mas suporta uma gama ***vasta*** de chips e pode programar dispositivos de até 40 pinos, tais como [MCU](/cars/rom/mcu)s de fábrica. O software para Windows é muito polido e fácil de usar. É de longe o melhor gravador que utilizei até agora. A minha única queixa real é que por vezes é um pouco instável a programar chips AT[29 C256](/cars/diagnostics/29c256), mas não tenho a certeza se o meu chip está a ficar cansado ou se a culpa é do programador. Também estou a correr uma versão antiga do software. Muitas vezes pode ser encontrado mais barato no [eBay](http://www.ebay.com) - Dave
