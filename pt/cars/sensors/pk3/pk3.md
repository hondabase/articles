# PK3 ECU Technical Reference

The PK3 ECU was utilized in 1990–1991 Honda Prelude models equipped with the B20A DOHC non-VTEC engine.

## Overview
The PK3 is an OBD0-based engine control unit. It is designed for naturally aspirated, non-VTEC DOHC configurations.

> [!NOTE]
> This ECU utilizes electronic ignition timing control. Ensure all ignition components are calibrated to factory specifications when troubleshooting timing-related issues.

## Technical Specifications
*   **Engine:** B20A (DOHC, non-VTEC)
*   **Vehicle Application:** 1990–1991 Honda Prelude
*   **OBD Standard:** OBD0
