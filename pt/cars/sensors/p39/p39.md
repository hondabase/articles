# P39 ECU Technical Reference

The P39 ECU is an OBD1 engine control unit utilized in 1992–1995 JDM Honda Prelude BB4 vehicles equipped with the F22B DOHC non-VTEC engine.

## ECU Overview

The P39 is typically found in automatic transmission configurations. Conversion to manual transmission operation requires specific hardware modifications to the PCB.

> [!IMPORTANT]
> When converting an automatic P39 ECU for manual transmission use, the resistor configuration must be updated. Specifically, the bridge at **RP11** must be moved to **RP12**.
