# P54 ECU Technical Reference

The P54 is an OBD1 engine control unit (ECU) utilized in 1992–1995 JDM Honda Domani and 1.8L Accord vehicles equipped with the 1.8L B18B DOHC engine.

## ECU Overview

The P54 ECU manages fuel and ignition mapping for the B18B engine platform. It follows standard OBD1 architecture and is compatible with common OBD1 tuning hardware.
