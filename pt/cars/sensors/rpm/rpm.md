# RPM (Revolutions Per Minute) Calculation

Revolutions Per Minute (RPM) represents the rotational speed of the engine crankshaft. In Honda engine management systems, this value is derived from the Crankshaft Position (CKP) sensor.

## Technical Specifications

The engine control unit (ECU) calculates engine speed based on the signal frequency generated by the CKP sensor.

*   **Signal Source:** Crankshaft Position (CKP) Sensor.
*   **Pulse Frequency:** 2 pulses per 1 full crankshaft rotation.

> [!NOTE]
> The ECU interprets the time interval between these pulses to determine the instantaneous engine speed. Consistent signal integrity from the CKP sensor is critical for accurate ignition timing and fuel injection synchronization.

## Signal Characteristics

The CKP sensor provides a square wave signal to the ECU. Because the system generates two pulses per revolution, the ECU performs a calculation to convert the frequency of these pulses into a standard RPM value.

| Parameter | Specification |
| :--- | :--- |
| **Pulses per Revolution** | 2 |
| **Signal Type** | Square Wave |
| **Primary Use** | Engine Speed (RPM) & Ignition Timing |

> [!TIP]
> If the engine fails to start or exhibits erratic RPM readings, inspect the CKP sensor wiring and the air gap between the sensor and the reluctor wheel for debris or excessive clearance.
