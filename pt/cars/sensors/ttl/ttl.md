# TTL

A lógica Transístor-Transístor (Transistor Transistor Logic) usa níveis de sinalização de 0/+5v. É habitualmente usada por [MCU](/cars/rom/mcu)s para [comunicação série](/cars/tuning/serial-communication) nativa.
