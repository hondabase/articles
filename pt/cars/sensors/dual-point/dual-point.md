# Dual Point

[Dual Point](/cars/sensors/dual-point) é utilizado para referir os motores e [ECU](/cars/ecu/ecu)s [DPFI](/cars/sensors/dpfi) (Dual-Point Fuel Injection - Injeção de Ponto Duplo). É essencialmente a geração seguinte de carburadores. O combustível é injetado com 2 injetores no coletor comum. O cilindro que estiver no tempo de admissão aspira o combustível. Separação de combustível, má atomização e distribuição do combustível são apenas alguns dos problemas que os sistemas [DPFI](/cars/sensors/dpfi) apresentam, sendo essas as principais razões pelas quais debitam menos potência do que os seus irmãos [MPFI](/cars/sensors/mpfi). O software para as [ECU](/cars/ecu/ecu)s [DPFI](/cars/sensors/dpfi), como a PM5/PM9 e P04, não é desenvolvido pois estas são consideradas inúteis para modificações de potência e afinação (tuning). Considere fazer uma conversão para [MPFI](/cars/sensors/mpfi) se procura obter mais potência.
