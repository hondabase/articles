# Oki 66207 Microcontroller Technical Reference

The Oki 66207 is a microcontroller unit (MCU) utilized extensively in Honda OBD1 engine control units (ECUs) for Civic and Integra platforms. It is based on the NX8/200 processor architecture.

## Architecture Overview

The 66207 serves as the primary processing unit for OBD1 ECU operations, managing fuel injection, ignition timing, and sensor data acquisition.

> [!NOTE]
> The 66207 utilizes the Oki NX8/200 core. Developers working with custom code or data logging should ensure their toolchains support this specific instruction set architecture.

## Technical Specifications

| Feature | Specification |
| :--- | :--- |
| **Manufacturer** | Oki Semiconductor |
| **Core Architecture** | NX8/200 |
| **Primary Application** | Honda OBD1 ECUs |
| **Package Type** | QFP (Quad Flat Package) |

## Development Resources

For advanced diagnostics and custom firmware development, refer to the following technical documentation regarding the NX8/200 core and 66207 pinout configurations:

*   **Instruction Set:** Consult the NX8/200 programming manual for opcode references.
*   **Memory Mapping:** OBD1 ECUs typically map the 66207 internal ROM/RAM alongside external EPROM storage.

> [!IMPORTANT]
> When probing the 66207 in-circuit, ensure the use of a high-impedance logic analyzer to prevent signal degradation or accidental triggering of the watchdog timer.
