# Vácuo Absoluto

O vácuo absoluto só existe verdadeiramente no espaço profundo. É o estado em que um objeto é rodeado por absolutamente nada. Existe absolutamente zero pressão de gás num vácuo absoluto. Só é possível aproximar-se de um vácuo absoluto aqui na Terra, mas é um conceito muito importante para coisas como [Pressão Absoluta](/cars/diagnostics/absolute-pressure)
