# MCU

**MCU** (Microcontroller Unit) é um microcontrolador. É um pequeno [CPU](/cars/reference/cpu), por vezes com recursos integrados de [RAM](/cars/reference/ram), [ROM](/cars/tuning/rom), [EPROM](/cars/ecu/eprom), [EEPROM](/cars/diagnostics/eeprom), [FLASH](/cars/ecu/flash) ou [IO](/cars/sensors/io). A documentação sobre os MCUs dentro das [ECU](/cars/ecu/ecu)s Honda [também está disponível aqui](/cars/diagnostics/66k-assembler-docs).
