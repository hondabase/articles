# RAM

**RAM** (Random Access Memory) é um tipo de memória de acesso aleatório.

> [!NOTE]
> A RAM é utilizada para armazenar variáveis e outros dados que estão ativamente em uso pelo processador.

> [!IMPORTANT]
> A RAM é uma memória volátil, o que significa que o seu conteúdo é apagado quando perde a alimentação.
