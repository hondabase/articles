# Perfil de Came

Um perfil de came é o formato distinto de uma árvore de cames. O levantamento de came (Cam Lift), a duração de came (Cam Duration) e o cruzamento de cames (Cam Overlap) são geralmente as três grandezas medidas numa árvore de cames.
