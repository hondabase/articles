# User Preferences

Users can customize interface settings to improve their browsing and technical reference experience. 

> [!NOTE]
> Preference values are currently stored using browser cookies. Ensure that cookies are enabled in your browser settings to persist these configurations across sessions.

## Available Settings

The following preferences can be adjusted to tailor the platform's behavior:

* **Display Units:** Toggle between Metric and Imperial measurements for sensor data and tuning parameters.
* **Theme Selection:** Switch between Light and Dark mode interfaces.
* **Default ECU View:** Set the preferred default view for ECU pinout and wirelist documentation.

## Managing Preferences

To update your settings:

1. Navigate to the **User Profile** or **Settings** menu located in the top navigation bar.
2. Adjust the desired parameters.
3. Click **Save Preferences** to commit the changes to your local browser storage.

> [!IMPORTANT]
> Clearing your browser cache or deleting cookies will reset all user preferences to their default values.
